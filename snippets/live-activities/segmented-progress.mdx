### Segmented Progress Type

Use `segmented_progress` for work that moves through clear stages, like deployments, onboarding flows, backups, ETL pipelines, migrations, and AI agent runs.

#### Stream

Use a stable `stream_key` and send the latest step whenever the state changes.

<img
  className="image"
  src="https://cdn.activitysmith.com/features/update-live-activity.png"
  alt="Segmented Progress Live Activity stream"
/>

```bash cURL
curl -X PUT https://activitysmith.com/api/live-activity/stream/nightly-backup \
  -H "Authorization: Bearer $ACTIVITYSMITH_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "content_state": {
      "title": "Nightly Backup",
      "subtitle": "upload archive",
      "type": "segmented_progress",
      "number_of_steps": 3,
      "current_step": 2
    }
  }'
```
