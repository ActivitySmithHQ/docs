### Progress Type

Use `progress` for real-time progress with percentage, like tasks, backups, migrations, syncs, or uploads.

#### Stream

Use a stable `stream_key` and send the latest percentage whenever the state changes.

<img
  className="image"
  src="https://cdn.activitysmith.com/features/progress-live-activity.png"
  alt="Progress Live Activity stream"
/>

```bash cURL
curl -X PUT https://activitysmith.com/api/live-activity/stream/search-reindex \
  -H "Authorization: Bearer $ACTIVITYSMITH_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "content_state": {
      "title": "Search Reindex",
      "subtitle": "catalog-v2",
      "type": "progress",
      "percentage": 42
    }
  }'
```
