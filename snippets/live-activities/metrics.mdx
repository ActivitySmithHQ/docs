### Metrics Type

Use `metrics` for live percentage values that change often, like server CPU, memory usage, disk usage, or error rate. Each metric takes a `label`, numeric `value`, and optional `unit`.

#### Stream

Use a stable `stream_key` and send the latest values whenever they change. ActivitySmith starts the Live Activity on the first call and updates it on later calls.

<img
  className="image"
  src="https://cdn.activitysmith.com/features/metrics-live-activity-start.png"
  alt="Metrics Live Activity stream"
/>

```bash cURL
curl -X PUT https://activitysmith.com/api/live-activity/stream/prod-web-1 \
  -H "Authorization: Bearer $ACTIVITYSMITH_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "content_state": {
      "title": "Server Health",
      "subtitle": "prod-web-1",
      "type": "metrics",
      "metrics": [
        { "label": "CPU", "value": 9, "unit": "%" },
        { "label": "MEM", "value": 45, "unit": "%" }
      ]
    }
  }'
```

#### End Live Activity

When the tracked process is finished, call `DELETE` with the same `stream_key` to dismiss the Live Activity. You can include final values before it is removed.

```bash cURL
curl -X DELETE https://activitysmith.com/api/live-activity/stream/prod-web-1 \
  -H "Authorization: Bearer $ACTIVITYSMITH_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "content_state": {
      "title": "Server Health",
      "subtitle": "prod-web-1",
      "type": "metrics",
      "metrics": [
        { "label": "CPU", "value": 7, "unit": "%" },
        { "label": "MEM", "value": 38, "unit": "%" }
      ],
      "auto_dismiss_minutes": 2
    }
  }'
```
